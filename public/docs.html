API Documentation

Full REST API reference for Axexvx.link. Build powerful link shortening, analytics, and SMS campaigns programmatically.

Introduction

The Axexvx.link API is a RESTful interface for creating and managing short links, accessing analytics, managing SMS campaigns, and controlling custom domains.

Base URL

https://api.axexvx.link/v1

All responses are in JSON. All requests must use HTTPS.

Authentication

Authenticate using your API key in the Authorization header. Get your key from Settings → API in the dashboard.

# Include in every request header
Authorization: Bearer axvx_live_xxxxxxxxxxxxxxxxxxxx
⚠️ Keep your API key secret. Never expose it in client-side code or public repositories. Rotate immediately if compromised.

API Key Types

TypePrefixAccess Level
Live Keyaxvx_live_Full production access
Test Keyaxvx_test_Sandbox only, no real SMS
Read-Onlyaxvx_ro_GET requests only

Create Short Link

POST/links

Request Body

ParameterTypeRequiredDescription
urlstring✓Destination URL to shorten
aliasstring—Custom slug. Auto-generated if omitted.
domainstring—Custom domain (Pro+). Defaults to axexvx.link
expires_atISO 8601—Link expiration datetime
passwordstring—Password-protect the link
max_clicksinteger—Disable after N clicks
tagsstring[]—Tags for organization

Example Request

curl -X POST https://api.axexvx.link/v1/links \
  -H "Authorization: Bearer axvx_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/download/app.apk",
    "alias": "myapp-v2",
    "domain": "axexvx.link",
    "expires_at": "2025-12-31T23:59:59Z",
    "tags": ["apk", "android"]
  }'

Response

{
  "id": "lnk_9xKmP3aQ",
  "short_url": "https://axexvx.link/myapp-v2",
  "destination": "https://example.com/download/app.apk",
  "alias": "myapp-v2",
  "qr_code": "https://api.axexvx.link/v1/qr/myapp-v2.png",
  "clicks": 0,
  "expires_at": "2025-12-31T23:59:59Z",
  "created_at": "2025-06-01T10:23:00Z"
}

Bulk Create Links

POST/links/bulk

Create up to 1,000 short links in a single request. Useful for large campaigns and APK distribution.

{
  "links": [
    { "url": "https://example.com/page1", "tags": ["batch1"] },
    { "url": "https://example.com/page2", "alias": "page2-short" },
    { "url": "https://example.com/app.apk", "tags": ["apk"] }
  ]
}

Send SMS

POST/sms/send

Send SMS with an auto-shortened link. Requires Pro or Business plan.

{
  "to": ["+12025550123", "+442071234567"],
  "message": "Download our app: {short_link} — Reply STOP to opt out.",
  "url_to_shorten": "https://example.com/app.apk",
  "sender_id": "AXEXVX",
  "schedule_at": "2025-07-01T09:00:00Z"
}

Bulk SMS Campaign

POST/sms/bulk

Send personalized SMS to up to 100,000 recipients. Supports {name}, {short_link}, and custom variable interpolation.

{
  "campaign_name": "App Launch Q3",
  "message_template": "Hi {name}, download our app: {short_link}",
  "url_to_shorten": "https://example.com/app-v2.apk",
  "recipients": [
    { "phone": "+12025550101", "name": "Alice" },
    { "phone": "+442071234500", "name": "Bob" }
  ]
}

Webhooks

Receive real-time HTTP POST notifications when events occur on your links or SMS campaigns.

EventDescription
link.clickedA short link was visited
link.expiredA link reached its expiry or max clicks
sms.deliveredAn SMS was successfully delivered
sms.failedAn SMS delivery failed
campaign.completedBulk SMS campaign finished sending
🔒 All webhook payloads are signed with HMAC-SHA256. Always verify the X-Axexvx-Signature header before processing.

Node.js Example

import Axexvx from '@axexvx/sdk';

const client = new Axexvx('axvx_live_YOUR_KEY');

const link = await client.links.create({
  url: 'https://example.com/app.apk',
  alias: 'my-app',
  tags: ['apk', 'android']
});

console.log(link.short_url); // https://axexvx.link/my-app
